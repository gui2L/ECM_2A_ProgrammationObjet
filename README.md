# Description du répertoire
Travaux réalisés suivant le cours **Programmation Objet** à *Centrale Méditerranée*; composés de dossiers TPs réalisés en C/C++  

--- 

### Versions utilisées : 
- **gcc.exe** (Rev8, Built by MSYS2 project) → `15.2.0`
- **g++.exe** (Rev8, Built by MSYS2 project) → `15.2.0`

---

### Compilation ---> Execution :
(Pour le dossier **tp1_c**)

```bash 
gcc fonctions.c main.c -o programme.exe  
``` 

(Pour les autres dossiers **tp.._..**)     

```bash 
g++ *cpp -o programme.exe 
```    
  
---> 
```bash 
./programme
```
